# OAI Project Generator

Welcome to the OAI Project Generator! This tool makes it easy to design and create powerful AI agent systems without writing any boilerplate code. Use our simple step-by-step wizard to build your architecture, and we'll package it all up for you in a ready-to-run project.

**Access the tool here**: [**Template Generator**](/generator)

## What is This Tool?

This is a visual builder that helps you create two main types of AI projects:

1.  **AI Agent Architectures**: Design complex systems where multiple AI agents work together to solve problems. You can create a single powerful agent or a team of specialized agents that collaborate, supervised by a manager agent.
2.  **MCP Servers**: Build simple, standalone servers that provide tools or data (like a calculator or a customer database) that your AI agents can use.

## Key Features

- **Simple Step-by-Step Process**: Our guided wizard walks you through every choice, from naming your project to configuring advanced features.
- **Build Your Agent, Your Way**:
  - **Choose a Strategy**: Select from different collaboration patterns, like a supervised team (`supervisor`) or a free-form swarm of agents (`swarm`).
  - **Add Capabilities**: With a single click, you can give your agents powerful abilities:
    - **Memory**: Allow your agent to remember past conversations.
    - **Knowledge**: Connect your agent to documents and data sources so it can answer specific questions.
    - **Tools & Skills**: Grant your agent the ability to use custom tools (like a web search) or perform specialized skills.
    - **Safety Rules**: Enable pre-built safety guardrails to ensure your agent's responses are secure and on-topic.
- **Live Preview**: As you make selections, you can see a live preview of the configuration file that will be generated, so you always know what's happening under the hood.
- **Instant Project Download**: Once you're done, just click "Generate Zip." You'll get a complete, runnable Python project with all your configurations, ready for you to customize further or deploy.

## How to Use It

Using the generator is as simple as 1-2-3:

1.  **Configure**: Follow the on-screen steps to define your project's details, add agents or servers, and select their capabilities.
2.  **Review**: On the final step, review the summary and the generated configuration files to make sure everything is perfect.
3.  **Download**: Click the "Download Architecture Zip" button to get your complete project.

That's it! You now have a fully functional starting point for your next-generation AI application.
