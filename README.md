# Docusaurus Demo Site

This project is a demo documentation site built with [Docusaurus](https://docusaurus.io/).

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm start
    ```
    This will open the site in your browser at `http://localhost:3000`.

## Project Structure

-   `docusaurus.config.js`: The configuration file for the site.
-   `docs/`: Contains the documentation markdown files.
-   `blog/`: Contains the blog markdown files.
-   `src/`: Contains the React components and pages.
-   `static/`: Contains static assets like images.
-   `sidebars.js`: Configures the sidebar structure for the docs.
